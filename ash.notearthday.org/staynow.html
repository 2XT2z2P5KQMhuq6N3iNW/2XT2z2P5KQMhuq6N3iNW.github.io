How can you contribute to or help with the a.s.h website?

Actually, you can't. This website is being maintained in a frozen status and is not accepting any more submissions. This link is being maintained for historical purposes only.

 

  • Do you want to contribute something to the a.s.h WWW?
  • Are you an a.s.h poster who wants to put your WWW address on the list?

If so, then send your contributions/WWW addresses/whatever to "ash@xanthia.com".

Basically, these pages are nothing without your contributions, so please send things in....

See below for more specific details

Getting yourself mentioned...

If you have a WWW page address, send the URL address using the above e-mail link.

If you are a poster on a.s.h and you do not have a WWW page but want a mention send either a text-only or html file with some information about yourself, and possibly a digitized picture of yourself. 

Getting yourself mentioned if you are anonymous....

If you are an anonymous poster in ashspace, you can still send in information if you wish to. Your name will simply not be given out.

Contributing a poem/song/quote...

If you want a poem, song or quote on the page, send it via e-mail to the above e-mail address. If you can, make it available on the WWW and send the URL address.

If you wish to remain anonymous, send your work, via e-mail to the above address, stating you wish to remain anonymous.

If the poem/song/quote is by another poster, please have the other poster send it in so that the webspinner knows that the webspinner has the person's permission.

Contributing a memorable post from a.s.h....

If the post is yours, please send it in along with how you wish to be credited for it (name, nick, anon).  If it is by another poster, the webspinner will have to check with the poster to make sure it is all right to put it on the web.
